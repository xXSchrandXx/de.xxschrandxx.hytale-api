# General
## Description
This plugin is an interface between other plugins and Hytale servers. Hytale API is mostly relevant to developers. It allows you to response to [WSC-Bridge](https://github.com/xXSchrandXx/WSC-Minecraft-Bridge).
## Requirements
[WSC-Bridge](#links) installed on your Bukkit- / Spigot- / BungeeCord-Server.
