# Setting up Hytale-API
[German](#German) | [English](#English)

## English
1. Install [Hytale-API](https://www.woltlab.com/pluginstore/file/TODO/) via the store code, via the package search or as a file upload.
2. Add your servers in the ACP via `Configuration > Hytale > Servers > +`.
3. Install [WSC-Bridge](https://github.com/xXSchrandXx/WSC-Bridge/blob/main/Setup.md) on your Hytale server.
